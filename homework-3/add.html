WebP

- Минусы: поддерживается не всеми браузерами (но это мне, кажется, вопрос времени); подходит не для всех картинок, например, для естественных фото (пластиковый вид, измение цветового профиля)
+ Плюсы: хорошее сжатие практически без потери качества; поддерживает прозрачность (отлично подходит для png картинок и статических gif)
